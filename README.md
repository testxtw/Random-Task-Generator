# Random-Task-Generator
GUI-приложение «Random Task Generator» с использованием библиотеки random, сохранением истории и Git.

## Автор: Шлемова Анастасия Николаевна

## Описание:
Генератор случайных задач — это простое приложение с графическим интерфейсом, которое помогает выбрать случайную задачу из списка (например, для саморазвития или отдыха). Можно добавлять свои задачи, фильтровать их по типу (учёба, спорт, работа), сохранять историю в файл и загружать её обратно.

## Технологии:
- Python 3.x.
- Tkinter (GUI).
- Random (генерация).
- JSON (сохранение истории).
- Git (контроль версий).

## Запуск:
1. Установите Python 3.x.
2. Установите необходимые библиотеки (если требуется):
   ```bash
   pip install -r requirements.txt  # (если есть зависимости)
   ```
3. Запустите приложение:
   ```bash
   python main.py
   ```
4. Используйте интерфейс:
   - Кнопка «Сгенерировать задачу» — выбирает случайную задачу.
   - Введите новую задачу и выберите тип — добавьте её в список.
   - Используйте фильтр по типу для просмотра истории.
5. История сохраняется в tasks.json.
