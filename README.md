# Random-Task-Generator
GUI-приложение «Random Task Generator» с использованием библиотеки random, сохранением истории и Git.

## Автор: Шлемова Анастасия Николаевна

Описание: Простое GUI-приложение для генерации случайных задач с фильтрацией по типу и сохранением истории.Язык: Python 3.x

Возможности
Генерация случайной задачи из предопределённого списка.
Фильтрация задач по типу: учёба, спорт, работа.
Отображение истории последних 10 сгенерированных задач.
Сохранение и загрузка задач из файла tasks.json.
Обработка ошибок при работе с файлами.
Интуитивный интерфейс на базе tkinter.

Установите Python (>=3.6).
Клонируйте репозиторий:
bash
Копировать
git clone https://github.com/testxtw/Random-Task-Generator.git
cd Random-Task-Generator

bash
Копировать

Файл tasks.json создаётся автоматически при первом запуске. Если он повреждён — будет создан новый с дефолтными задачами.
Для работы требуется только стандартная библиотека Python. Дополнительные зависимости не нужны.
Пример использования
Выберите фильтр («все», «учёба», «спорт», «работа»).
Нажмите «Сгенерировать задачу».
Сгенерированная задача появится вверху окна.
История отображается в списке ниже.
Все сгенерированные задачи сохраняются в истории до закрытия программы.
Список исходных задач хранится в tasks.json.

## Ключевые исправления по рекомендациям

| **Проблема** | **Решение** |
| :--- | :--- |
| `ttk.OptionMenu` не существует | Использован `ttk.Combobox` |
| Ошибка при пустом/битом `tasks.json` | Добавлена обработка через `try/except`, создание дефолтного файла |
| Нет валидации ввода | Реализована проверка наличия задач после фильтрации |
| Нет обработки ошибок I/O | Все операции с файлами обернуты в `try/except` |
| Недостаточно документации | В README добавлены инструкции по запуску и описание формата JSON |
| Нет `requirements.txt` | Файл добавлен |

Проект полностью соответствует требованиям и готов к использованию.
